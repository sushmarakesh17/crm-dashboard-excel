# CRM Dashboard

## Project Overview

CRM Dashboard is an Excel-based analytics project designed to track customer relationship data, analyze sales performance, and generate business insights.

The dashboard helps businesses monitor key metrics and make data-driven decisions.

## Objectives

* Analyze customer relationship data
* Track sales and customer performance
* Create interactive business dashboards
* Generate actionable insights

## Technologies Used

* Microsoft Excel
* Google Sheets
* Data Analysis
* Dashboard Design

## Features

* Customer data analysis
* Sales performance tracking
* KPI monitoring
* Interactive charts and dashboards
* Business insights generation

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Analysis
4. KPI Calculation
5. Dashboard Creation
6. Insight Generation

## Key Metrics

* Total Customers
* Sales Performance
* Customer Trends
* Revenue Analysis
* Performance Indicators
